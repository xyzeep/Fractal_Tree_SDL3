## What is this?

A very simple implementation of Fractal Tree (aka Fractal Canopy) written in C using SDL3. The program recursively draws branches from a starting trunk, splitting each branch into two smaller branches at a fixed angle.

## Try it yourself!

1. Install **SDL3** on your system:

   Download SDL3 from the official release ([here](https://github.com/libsdl-org/SDL/))

2. Clone the repository:
   `git clone https://github.com/xyzeep/Fractal_Tree_SDL3.git`
3. Compile and Run:

   `gcc main.c -I[path/to/sdl/include] -L[path/to/sdl/lib] -lSDL3 -o main`

   `.\main.exe`
